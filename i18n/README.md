## Want to add a new language?

1. Copy `../README.md` to `i18n/README.{lang}.md`
2. Translate all human-readable text (keep code blocks, badges, and links as-is)
3. Add your language to the table above
4. Add a language switcher line to the top of your translated file
5. Open a PR with title: `i18n: add [Language] README`

### Language Code Reference

Use [ISO 639-1](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes) two-letter codes:
`ar` Arabic · `zh` Chinese · `es` Spanish · `fr` French · `de` German · `ja` Japanese · `ko` Korean · `pt` Portuguese · `ru` Russian · `hi` Hindi

### Translation Guidelines

- **Keep** all code blocks, badge URLs, and links unchanged
- **Translate** all headings, descriptions, table text, and prose
- **Preserve** the original document structure and emoji
- **Add** a language switcher at the top linking back to English and other translations
- **Update** the skill count badges to match the current count in the English README
