## 문제점

AI 에이전트를 만드는 모든 개발자가 동일한 스킬을 처음부터 다시 발견합니다.

누군가는 RAG를 어렵게 배웁니다. 또 다른 사람은 새벽 2시에 메모리 인젝션을 깨닫습니다. 세 번째 사람은 ReAct와 LATS를 비교하는 데 일주일을 보내고 결과를 공유하지 않습니다. 네 번째 사람은 당신이 지난달에 겪은 것과 똑같은 실패 패턴을 마주칩니다.

**그 집단적 지식은 Slack 스레드, 비공개 저장소, Twitter 북마크 속으로 사라지고 있습니다.**

Skills Tree가 이 문제를 해결합니다.

---

## 무엇인가요?

**Skills Tree는 AI 에이전트 능력을 위한 공유 운영체제입니다.**

에이전트가 할 수 있는 모든 것의 살아있는, 버전 관리된, 커뮤니티 기반 인덱스 — 동작하는 코드, 실제 벤치마크, 실패 모드, 진화 역사와 함께 문서화됩니다. 모든 스킬은 프로덕션 준비 완료. 모든 시스템은 스킬이 어떻게 결합되는지 보여줍니다. 모든 벤치마크는 재현 가능합니다.

목록이 아닙니다. 인프라입니다.

## 🤝 기여 방법

| 유형 | 내용 | PR 제목 형식 |
|---|---|---|
| **새 스킬** | 아직 인덱싱되지 않은 능력 | `feat: add [skill] to [category]` |
| **스킬 업그레이드** | v1→v2로 향상된 콘텐츠 | `improve: [skill] — v1→v2` |
| **벤치마크** | 실제 숫자로 직접 비교 | `benchmark: [skill-a] vs [skill-b]` |
| **시스템 / 블루프린트** | 멀티 스킬 워크플로우 또는 아키텍처 | `system: add [name]` |
