## Проблема

Каждый разработчик ИИ-агентов заново открывает одни и те же навыки с нуля.

Кто-то учится использовать RAG на собственных ошибках. Другой разбирается с инъекцией памяти в 2 часа ночи. Третий тратит неделю на сравнение ReAct и LATS — и никогда не делится результатами. Четвёртый натыкается на те же ошибки, которые вы уже совершили в прошлом месяце.

**Эти коллективные знания исчезают в тредах Slack, приватных репозиториях и закладках Twitter.**

Skills Tree решает эту проблему.

---

## Что это такое?

**Skills Tree — это общая операционная система для возможностей ИИ-агентов.**

Живой, версионированный, поддерживаемый сообществом индекс всего, что может делать агент — с документацией, включающей рабочий код, реальные бенчмарки, режимы отказов и историю развития. Каждый навык готов к продакшену. Каждая система показывает, как навыки сочетаются. Каждый бенчмарк воспроизводим.

Это не список. Это инфраструктура.

## 🤝 Как внести вклад

| Тип | Что это | Формат заголовка PR |
|---|---|---|
| **Новый навык** | Способность, ещё не включённая в индекс | `feat: add [skill] to [category]` |
| **Улучшение навыка** | Обновление v1→v2 с лучшим содержимым | `improve: [skill] — v1→v2` |
| **Бенчмарк** | Прямое сравнение с реальными числами | `benchmark: [skill-a] vs [skill-b]` |
| **Система / Чертёж** | Мультинавыковой рабочий процесс или архитектура | `system: add [name]` |
