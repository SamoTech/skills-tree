# Supported Frameworks & Models

A list of frameworks, platforms, and models referenced throughout the Skills Tree.

## Agent Frameworks

| Framework | Language | Description | Link |
|---|---|---|---|
| **LangChain** | Python / JS | Most popular agent framework, tool-use, RAG | [langchain.com](https://langchain.com) |
| **LangGraph** | Python | Graph-based multi-agent orchestration | [langchain.com/langgraph](https://langchain.com/langgraph) |
| **AutoGen** | Python | Microsoft's multi-agent conversation framework | [microsoft.github.io/autogen](https://microsoft.github.io/autogen) |
| **CrewAI** | Python | Role-based multi-agent crews | [crewai.com](https://crewai.com) |
| **Semantic Kernel** | Python / C# | Microsoft's SDK for LLM integration | [learn.microsoft.com](https://learn.microsoft.com/semantic-kernel) |
| **Haystack** | Python | NLP pipelines and RAG | [haystack.deepset.ai](https://haystack.deepset.ai) |
| **Camel** | Python | Role-playing autonomous agents | [camel-ai.org](https://camel-ai.org) |
| **MetaGPT** | Python | Software company simulation with agents | [deepwisdom.ai](https://deepwisdom.ai) |
| **Agno** | Python | High-performance agentic framework | [agno.com](https://agno.com) |
| **Pydantic AI** | Python | Type-safe agent framework | [ai.pydantic.dev](https://ai.pydantic.dev) |
| **DSPy** | Python | Programming language for LLM pipelines | [dspy.ai](https://dspy.ai) |
| **Smolagents** | Python | HuggingFace's lightweight agent library | [huggingface.co](https://huggingface.co/docs/smolagents) |

| **Vertex AI Agents** | Python | Google Cloud agent platform | [cloud.google.com](https://cloud.google.com/vertex-ai/generative-ai/docs/agents) |
| **Bedrock Agents** | Python | AWS agent platform | [aws.amazon.com](https://aws.amazon.com/bedrock/agents) |
