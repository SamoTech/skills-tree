# Glossary

Key terms used throughout the Skills Tree.

| Term | Definition |
|---|---|
| **Agent** | An AI system that perceives its environment and takes actions to achieve goals |
| **Skill** | A discrete, reusable capability an agent can invoke |
| **Tool** | An external function or API the agent can call |
| **Action** | A concrete operation performed in the world (click, type, API call) |
| **Perception** | The ability to read/sense inputs (text, images, files, screen) |
| **Reasoning** | The ability to think, plan, and decide (CoT, ReAct, MCTS) |
| **Memory** | Mechanisms for storing and retrieving information across turns |
| **Orchestration** | Coordinating multiple agents or tools to complete a task |
| **Computer Use** | The ability to interact with GUIs, OS, and desktop apps |
| **Grounding** | Connecting language understanding to real-world referents |
| **ReAct** | Reasoning + Acting — interleaved thought and action steps |
| **CoT** | Chain of Thought — step-by-step internal reasoning |
| **RAG** | Retrieval-Augmented Generation — fetching context before generating |
| **Tool Call** | An invocation of an external function by the model |
| **Multimodal** | Capable of handling multiple modalities (text, image, audio, video) |
| **Sandbox** | An isolated execution environment for running untrusted code |
| **LLM** | Large Language Model — the core reasoning engine of most agents |
| **Embedding** | A vector representation of text/data for semantic search |
| **Vector Store** | A database optimized for similarity search over embeddings |
| **Agentic Loop** | The perceive → reason → act → observe cycle agents run in |
| **Function Calling** | Structured output from LLMs to invoke external functions |
| **Prompt** | The input instruction given to an LLM |
| **Context Window** | The maximum token length an LLM can process at once |
| **Fine-tuning** | Training a model further on task-specific data |
| **Zero-shot** | Performing a task with no examples provided |
| **Few-shot** | Performing a task with a few examples in the prompt |
