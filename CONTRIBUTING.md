# Contributing to Skills Tree

Thank you for helping grow the Skills Tree! 🌱

## How to Add a Skill

1. **Find the right category** in `skills/` — browse the folders to find where your skill fits.
2. **Copy the template** from `meta/skill-template.md`.
3. **Fill in all fields** — name, description, inputs, outputs, example, frameworks.
4. **Add it to the category's README.md** skill list table.
5. **Open a pull request** with the title: `feat: add [skill-name] to [category]`

## Skill Quality Guidelines

- ✅ Real capability that an existing agent/model can perform
- ✅ Clear, concise description (1-2 sentences)
- ✅ At least one concrete example
- ✅ Linked to a framework or model that implements it
- ❌ No speculative or hypothetical future skills
- ❌ No duplicates — search first

## Category Proposal

If your skill doesn't fit any existing category, open an issue with label `new-category` and describe the skill family.
