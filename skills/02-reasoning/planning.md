# Planning

**Category:** `reasoning`  
**Skill Level:** `advanced`  
**Stability:** `stable`
**Added:** 2025-03

### Description

Break a high-level goal into an ordered sequence of sub-tasks, estimate dependencies, and produce an executable plan.

### Example

```
Goal: Build and deploy a REST API
Plan:
1. Define API endpoints and data models
2. Set up project structure and dependencies
3. Implement each endpoint
4. Write unit tests
5. Containerize with Docker
6. Deploy to cloud (Railway / Fly.io)
```
