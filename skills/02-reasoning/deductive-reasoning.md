# Deductive Reasoning

**Category:** `reasoning`  
**Skill Level:** `intermediate`  
**Stability:** `stable`
**Added:** 2025-03

### Description

Derive logically certain conclusions from general premises using formal or informal logic.

### Example

```
Premise 1: All MCP servers expose tools.
Premise 2: GitHub MCP is an MCP server.
Conclusion: Therefore, GitHub MCP exposes tools.
```
