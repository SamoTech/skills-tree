# Analogical Reasoning

**Category:** `reasoning`  
**Skill Level:** `intermediate`  
**Stability:** `stable`
**Added:** 2025-03

### Description

Solve problems or generate explanations by drawing structural parallels between a familiar domain and an unfamiliar one.

### Example

```
Problem: Explain how RAG works.
Analogy: It's like an open-book exam — the model doesn't memorize everything;
instead it looks up relevant notes before answering.
```
