# Commonsense Reasoning

**Category:** `reasoning`  
**Skill Level:** `intermediate`  
**Stability:** `stable`

### Description

Apply everyday world knowledge that is not explicitly stated — physical, social, and temporal intuitions.

### Example

```
Q: If I leave ice cream outside on a hot day, what happens?
A: It melts, because ice cream is frozen and heat causes it to liquefy.
```
