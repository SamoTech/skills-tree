# Risk Assessment

**Category:** `reasoning`  
**Skill Level:** `advanced`  
**Stability:** `stable`
**Added:** 2025-03

### Description

Identify potential negative outcomes of a plan or action, estimate their likelihood and severity, and suggest mitigations.

### Example

```
Action: Deploy to production without staging test
Risk: HIGH probability, CRITICAL severity (full outage possible)
Mitigation: Run smoke tests on staging first, then deploy with feature flag.
```
