# Abductive Reasoning

**Category:** `reasoning`  
**Skill Level:** `advanced`  
**Stability:** `stable`

### Description

Infer the most plausible explanation for an observation given incomplete information — reasoning to the best explanation.

### Example

```
Observation: The floor is wet.
Possible causes: rain came in, pipe burst, someone spilled water, cleaning.
Best explanation given context (kitchen, sunny day): someone spilled water.
```
