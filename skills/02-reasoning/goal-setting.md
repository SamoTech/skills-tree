# Goal Setting

**Category:** `reasoning`  
**Skill Level:** `intermediate`  
**Stability:** `stable`
**Added:** 2025-03

### Description

Decompose a high-level objective into clear, measurable, achievable sub-goals that guide agent execution.

### Example

```
High-level goal: "Build a skills-tree repository"
Sub-goals:
  1. Define taxonomy of skill categories
  2. Create README and folder structure
  3. Write individual skill files
  4. Add contribution guidelines
```
