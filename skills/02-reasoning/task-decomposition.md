# Task Decomposition

**Category:** `reasoning`  
**Skill Level:** `intermediate`  
**Stability:** `stable`

### Description

Split a complex task into smaller, independent or sequential sub-tasks that can each be solved or delegated separately.

### Example

```
Task: Write a research report on climate change
Sub-tasks:
- Search for recent statistics
- Identify top causes
- Summarize mitigation strategies
- Write introduction
- Write conclusion
- Format and export as PDF
```
