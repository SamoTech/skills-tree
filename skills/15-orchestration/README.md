# 🎼 Orchestration Skills

Orchestration skills enable agents to **coordinate, delegate, and manage multi-agent workflows** at scale.

## Skills in This Category

| Skill | Level | Description |
|---|---|---|
| [Parallel Task Execution](parallel-execution.md) | Advanced | Run multiple agent tasks concurrently |
| [Sequential Workflow](sequential-workflow.md) | Intermediate | Execute tasks in ordered steps |
| [Conditional Branching](conditional-branching.md) | Intermediate | Take different paths based on conditions |
| [Retry with Backoff](retry-backoff.md) | Intermediate | Retry failed tasks with exponential backoff |
| [Subagent Spawning](subagent-spawning.md) | Advanced | Create and manage child agent instances |
| [Agent Communication](agent-communication.md) | Advanced | Pass messages between agents |
| [Shared Memory / Blackboard](shared-memory.md) | Advanced | Agents share a common memory space |
| [Task Queue Management](task-queue.md) | Advanced | Manage ordered queues of pending tasks |
| [Consensus Voting](consensus-voting.md) | Advanced | Multiple agents vote on a decision |
| [Role Assignment](role-assignment.md) | Intermediate | Assign specialized roles to agents |
| [Workflow State Machine](state-machine.md) | Advanced | Model workflows as finite state machines |
| [Event-Driven Triggers](event-triggers.md) | Advanced | Trigger agents in response to events |
| [Hierarchical Agent Tree](hierarchical-tree.md) | Advanced | Manage tree of orchestrator and worker agents |
| [Budget / Token Management](budget-management.md) | Intermediate | Track and limit agent token consumption |
| [Logging and Observability](logging-observability.md) | Intermediate | Trace and monitor agent executions |
| [Agent Handoff](agent-handoff.md) | Advanced | Transfer task context between agents cleanly |
