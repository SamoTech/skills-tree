# Argument Construction

**Category:** `communication`  
**Skill Level:** `advanced`  
**Stability:** `stable`

### Description

Build a well-structured, evidence-backed argument or position on a topic with logical coherence.

### Example

```
Claim: AI agents should require human approval for destructive actions.
Support:
  P1: Irreversible actions (file deletion, DB writes) can cause data loss.
  P2: Agents can misinterpret ambiguous instructions.
  P3: Human oversight maintains accountability and trust.
Conclusion: A confirmation step prevents costly mistakes.
```
