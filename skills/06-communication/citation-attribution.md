# Citation & Attribution

**Category:** `communication`  
**Skill Level:** `intermediate`  
**Stability:** `stable`

### Description

Cite sources accurately and attribute information to its origin to maintain credibility and avoid misinformation.

### Example

```
Fact: "GPT-4 achieves human-level performance on professional exams."
Citation: OpenAI. (2023). GPT-4 Technical Report. https://arxiv.org/abs/2303.08774
```
