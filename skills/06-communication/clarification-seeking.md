# Clarification Seeking

**Category:** `communication`  
**Skill Level:** `intermediate`  
**Stability:** `stable`
**Added:** 2025-03

### Description

Identify ambiguities in a user request and ask a focused, targeted question to resolve them before proceeding.

### Example

```
User: "Update the file."
Agent: "Which file should I update, and what changes should I make to it?"

User: "Fix the bug."
Agent: "Could you share the error message or the file where the bug appears?"
```
