# 💬 Communication Skills

Communication skills enable agents to **express, translate, summarize, and converse** in natural language across styles, languages, and formats.

## Skills in This Category

| Skill | Level | Description |
|---|---|---|
| [Summarization](summarization.md) | Basic | Condense long content into key points |
| [Translation](translation.md) | Basic | Translate text between languages |
| [Paraphrasing](paraphrasing.md) | Basic | Rewrite content in different words |
| [Tone Adjustment](tone-adjustment.md) | Intermediate | Change tone (formal, casual, assertive) |
| [Question Answering](question-answering.md) | Basic | Answer factual or contextual questions |
| [Clarification Seeking](clarification-seeking.md) | Intermediate | Ask follow-up questions to reduce ambiguity |
| [Instruction Following](instruction-following.md) | Basic | Execute step-by-step user instructions |
| [Report Writing](report-writing.md) | Intermediate | Compose structured written reports |
| [Email Drafting](email-drafting.md) | Basic | Write professional emails |
| [Argument Construction](argument-construction.md) | Advanced | Build structured arguments with evidence |
| [Debate / Counterargument](debate.md) | Advanced | Present opposing viewpoints |
| [Multilingual Output](multilingual-output.md) | Intermediate | Generate text in multiple languages |
| [Structured Output Formatting](structured-output.md) | Intermediate | Output JSON, XML, Markdown, tables |
| [Citation and Attribution](citation-attribution.md) | Intermediate | Properly cite sources in text |
| [Persona Adoption](persona-adoption.md) | Advanced | Speak as a specific character or expert role |
