# Screen Reading

**Category:** `perception`  
**Skill Level:** `intermediate`  
**Stability:** `stable`

### Description

Capture a screenshot of the current desktop or browser and parse its content — UI elements, text, buttons, and layout.

### Example

```python
import pyautogui
screenshot = pyautogui.screenshot()
# Pass to vision model for parsing
```

- Anthropic Computer Use
- OpenAI CUA
- `pyautogui`, `mss`
