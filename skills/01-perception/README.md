# 👁️ Perception Skills

Perception skills enable agents to **sense, read, and interpret inputs** from their environment — text, images, files, screens, sensors, and more.

## Skills in This Category

| Skill | Level | Description |
|---|---|---|

| [OCR](ocr.md) | Intermediate | Extract text from images and scanned documents |
| [Screen Reading](screen-reading.md) | Intermediate | Capture and parse a desktop/browser screenshot |
| [Audio Transcription](audio-transcription.md) | Intermediate | Convert speech to text |

| [URL/DOM Inspection](url-dom-inspection.md) | Intermediate | Inspect live web page DOM structure |
