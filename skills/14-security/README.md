# 🔒 Security & Safety Skills

Security skills ensure agents **operate safely, respect permissions, avoid harmful actions**, and protect systems they interact with.

## Skills in This Category

| Skill | Level | Description |
|---|---|---|
| [Sandboxed Code Execution](sandboxed-execution.md) | Intermediate | Run code in isolated, restricted environments |
| [Permission Checking](permission-checking.md) | Intermediate | Verify agent has permission before taking action |
| [Secret Scanning](secret-scanning.md) | Intermediate | Detect API keys, passwords in code/files |
| [Input Sanitization](input-sanitization.md) | Intermediate | Sanitize user inputs to prevent injection |
| [Rate Limiting Awareness](rate-limiting.md) | Basic | Respect API rate limits and retry correctly |
| [Audit Logging](audit-logging.md) | Intermediate | Log all agent actions for traceability |
| [Harm Detection](harm-detection.md) | Advanced | Detect potentially harmful instructions |
| [Privacy Preservation](privacy-preservation.md) | Advanced | Avoid exposing PII in outputs or logs |
| [Rollback / Undo](rollback-undo.md) | Advanced | Reverse actions if something goes wrong |
| [Human-in-the-Loop Escalation](human-in-loop.md) | Advanced | Escalate risky decisions to human review |
