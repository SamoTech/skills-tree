# ⚡ Action Execution Skills

Action execution skills enable agents to **take concrete actions** in the world — sending messages, writing files, calling APIs, and interacting with environments.

## Skills in This Category

|---|---|---|

| [Shell Command Execution](shell-command.md) | Intermediate | Run shell/bash commands |
| [HTTP Request](http-request.md) | Intermediate | Make GET/POST/PUT/DELETE HTTP calls |
| [Email Sending](email-sending.md) | Intermediate | Compose and send emails |

| [Clipboard Operations](clipboard-ops.md) | Basic | Copy/paste to/from system clipboard |
