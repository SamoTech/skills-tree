# Mouse Input

**Category:** `action-execution`  
**Skill Level:** `basic`  
**Stability:** `stable`
**Added:** 2025-03

### Description

Simulate mouse movements, clicks, right-clicks, and double-clicks at specified screen coordinates.

### Example

```python
import pyautogui
pyautogui.moveTo(960, 540, duration=0.3)
pyautogui.click()
pyautogui.rightClick(200, 300)
```
