## 📖 What Is This?

**Skills Tree** is a living catalog of every capability, skill, tool, mode, and action an AI agent or computer-use model can perform — from basic I/O to complex multi-agent orchestration.

Use it to:
- 🔍 Discover what AI agents can actually do
- 🛠️ Design agent systems and pick the right tools
- 📚 Learn about agentic frameworks and capabilities
- 🤝 Contribute new skills you've discovered

## 🚀 Quick Start

Browse by category in `/skills/` or search using GitHub's code search.

```bash
git clone https://github.com/SamoTech/skills-tree.git
cd skills-tree
# Browse any category
cat skills/05-code/README.md
```
